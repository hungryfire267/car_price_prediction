# Car Price Prediction Project

## Introduction
This project focuses on building a machine learning model to predict car prices based on various vehicle features. The goal is to analyse historical car data and generate accurate price predictions using regression techniques.

## Overview
Accurate car price estimation helps buyers, sellers, and dealerships make informed decisions.

This notebook walks through:

- Data cleaning and preprocessing
- Feature engineering
- Exploratory Data Analysis (EDA)
- Model training and evaluation
- Price prediction
